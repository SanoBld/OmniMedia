MediaTools

Téléchargeur + convertisseur + compresseur — tout se passe dans le navigateur, rien n'est uploadé

Utilise cobalt.tools — open source, sans pub, sans compte. En cas d'échec, cliquez sur "Ouvrir dans Cobalt" ou réessayez après quelques secondes.
Propulsé par FFmpeg.wasm — la conversion s'effectue entièrement dans le navigateur, rien n'est uploadé. Premier chargement : ~25 Mo (10–30 secondes). Fichiers >500 Mo peuvent être lents.
